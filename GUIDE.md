# ⚡️ LazyVim & Go 开发终极指南 (AI 增强版)

本配置专为追求极致效率和现代视觉的开发者打造，完美适配 Neovide 和 Kitty。

## 🤖 AI 智能辅助 (三剑客)
系统已配置三款主流 AI，默认处于“多轨并行”状态。**如需切换，请修改 `lua/plugins/` 下对应文件的 `enabled = true/false`。**

### 1. GitHub Copilot (稳定首选)
- **快捷键**: `<Tab>` 全量接受 / `Alt + l` 接受一个词。
- **面板**: `<leader>cp` 呼出多方案对比。
- **配置**: `lua/plugins/copilot.lua`。

### 2. Supermaven (速度之王)
- **快捷键**: `<Tab>` 接受补全 / `Alt + l` 接受一个词。
- **状态**: `:SupermavenStatus`。
- **配置**: `lua/plugins/supermaven.lua`。

### 3. Fitten Code (国产免费)
- **快捷键**: `<Tab>` 接受全部 / `Alt + l` 接受下一行。
- **登录**: `:Fitten login`。
- **配置**: `lua/plugins/fittencode.lua`。

## 🎨 视觉与 UI (透明极致)
- **背景透明**: 已启用 `transparent.nvim`，完美穿透侧边栏和浮窗。
- **标签页 (Bufferline)**: 
    - 顶部显示序号 (1, 2, 3...)。
    - **`Alt + 1..9`**: 快速跳转标签。
    - **`Alt + w`**: 关闭当前标签。
- **输入法自动切换**: 
    - 启动、聚焦、退出插入模式时，**全自动、异步、零延迟**切回英文。

## 🚀 极速开发指令

### Go 语言专属
- **`<leader> g r`**: 运行项目 (`go run .`)。
- **`<leader> g f`**: 运行当前文件。
- **`F2`**: 智能重命名变量 (LSP)。
- **`F5 / F10 / F11`**: 启动调试 / 单步跳过 / 进入。

### 文件与会话
- **`<leader> q s`**: **恢复上次会话** (快速找回所有打开的标签页)。
- **`<leader> e`**: 打开文件树。
- **`.` (文件树内)**: 切换显示/隐藏文件。

### 编辑神技
- **`jj`**: 插入模式秒退 (代替 Esc)。
- **`yiw`**: 快速复制光标下的单词 (不进入选择模式)。
- **`xx`**: **黑洞删除** (不污染剪贴板删除整行)。
- **`Shift + Enter`**: 插入模式下快速向上开新行。
- **`ca(`**: 修改括号内的所有内容。
