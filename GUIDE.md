# LazyVim & Go 开发指南 (Ultimate Edition)

这份指南涵盖了你的 Neovim 定制环境的所有核心操作，适用于 Neovide 和 Kitty。

## 1. 核心快捷键 (通用)

### 🚀 基础操作
| 按键 | 功能 |
| --- | --- |
| **`Ctrl + s`** | 保存文件 |
| **`u`** | 撤销 (Undo) - 需在 Normal 模式 |
| **`Ctrl + r`** | 重做 (Redo) - 需在 Normal 模式 |
| **`Alt + a`** | 全选 (Neovide 也支持 `Cmd+a`) |
| **`Alt + 1..9`** | 切换到第 N 个标签页 |
| **`Alt + w`** | 关闭当前标签页 |

### ⚡️ 编辑效率
| 按键 | 功能 |
| --- | --- |
| **`jj`** | 退出插入模式 (秒退) |
| **`Shift + Enter`** | (插入模式) 在**上方**插入新行 |
| **`Ctrl + Enter`** | (插入模式) 在**下方**插入新行 |
| **`x` / `xx`** | **黑洞删除** (只删不复制，不污染剪贴板) |
| **`<leader>h` / `l`** | 跳到 行首 / 行尾 |
| **`Ctrl + d` / `u`** | 向下/向上翻半页 (自动居中) |

### 🧩 高级编辑 (文本对象)
- **`yiw`**: **快速复制当前单词** (光标在单词内即可)。
- **`ci"`**: 修改引号内的内容 (直接删除并进入插入模式)。
- **`ca(`**: 修改括号及其内容。
- **`gcc`**: 注释/取消注释当前行。

## 2. Go 开发全流程

### 运行与调试
- **`<leader> g r`**: 运行整个项目 (`go run .`)。
- **`<leader> g f`**: 运行当前文件。
- **`F5`**: 继续调试 / 启动调试。
- **`F10`**: 单步跳过 (Step Over)。
- **`F11`**: 单步进入 (Step Into)。
- **`F2`**: 重命名变量 (LSP Rename)。

### AI 辅助 (Copilot)
- **`<Tab>`**: 接受 AI 的灰色建议。
- **`Alt + p`**: 打开 Copilot 面板 (查看多行/多个建议)。

---

## 3. 文件管理
- **`<leader> e`**: 打开/关闭文件树 (Snacks Explorer)。
- **`Shift + h / l`**: 左右切换标签页 (备选方案)。
- **`<leader> f f`**: 查找文件 (支持隐藏文件)。
- **`<leader> /`**: 全局搜索内容。

---

## 4. 特殊说明
- **复制/粘贴**:
  - 在 **Neovide** 中：使用 `Cmd + C` / `Cmd + V`。
  - 在 **Kitty** 中：需依赖终端自身的复制粘贴键，或配置 `kitty.conf`。
- **精简模式**:
  - 按键 `a` 已被禁用。请使用 `i` (插入), `A` (行尾插入) 或 `o/O` (换行插入)。
